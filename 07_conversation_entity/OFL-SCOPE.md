## Applies to

| Path | Material |
|------|----------|
| `fonts/` | All font software (`.ttf`, `.otf`, `.woff`, `.woff2`) and `FONTLOG.md` |

## You may

- Use, embed, modify, and redistribute fonts with CIM materials
- Bundle fonts with MIT / CIM / WTFPL-licensed software in this repo
- Create documents (PDF, HTML) using OFL fonts — documents are **not** bound by OFL

## You must

- Keep fonts under OFL (not relicense font files under MIT/WTFPL alone)
- Include `LICENSE-OFL` or `OFL-NOTICE.txt` when redistributing font files
- Respect **Reserved Font Names** listed in `fonts/FONTLOG.md`

## Does not apply to

- **Helvetica canon** — archived under `fonts/CIM-Visualist/Helvetica/` (URW NimbusSans, see FONTLOG)
- ReportLab Standard 14 Helvetica/Courier names in PDF output (see `cim_visualist_typography.py`)
- Sketch imagery, Java source, tutorial prose (see MIT / CIM / WTFPL)
