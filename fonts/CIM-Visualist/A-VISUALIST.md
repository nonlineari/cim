## Archived substitute

System Helvetica may be withdrawn on some hosts. CIM archives **URW Nimbus Sans**
(`fonts/CIM-Visualist/Helvetica/NimbusSans/`) — the standard PDF/PostScript metric
compatible substitute — so **A. Visualist** deliverables keep the same rhythm.

ReportLab **Standard 14** names (`Helvetica`, `Helvetica-Bold`, …) remain the
runtime default in `cim_visualist_typography.py`; archived `.afm`/`.t1` files
are the offline preserve.
